# Personal-Expense-Tracker-with-Search-Category-Filter.
Built a desktop application using Tkinter that helps users track daily expenses efficiently. It includes search and category filtering features for better financial management.

Key Features:
- Add, view, and manage expenses
- Search functionality for quick lookup
- Category-based filtering (Food, Travel, etc.)
- Simple and clean GUI
- Lightweight and fast

Tools Used:
- Python
- Tkinter

Outcome:
- Helps users manage their spending and maintain financial discipline.
